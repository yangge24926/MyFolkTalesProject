# MyFolkTalesProject

一个用于创作中文民间故事、传说、怪谈与志异短中篇的仓库。

## 目标

- 每篇故事独立完结，不强制共享世界观。
- 单篇字数范围：10000-50000 字。
- 风格优先：有画面感、有因果、有余味，避免流水账与设定堆砌。

## 目录结构

- `stories/`：故事正文文件，建议一篇一个文件。
- `docs/`：创作流程、状态记录与模板。
- `.cursor/rules/`：面向 AI 协作的写作规则。

## 推荐命名

故事文件命名建议：

`stories/0001_故事标题.md`

首行建议：

`第1篇 故事标题`

## 快速开始

1. 先看：`world_setting.md`、`characters.md`、`plot_tracker.md`。
2. 根据 `docs/writing_steps.md` 的流程写第一稿。
3. 每写完一篇，更新 `plot_tracker.md` 的状态表。
